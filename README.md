# MAP

Move and Pivot


## INTRO

An adventure game.

## USAGE
On Linux or Mac:

    $ cd /path/to/this/directory
    $ chmod +x run-linux-or-mac.sh
    $ ./run-linux-or-mac.sh

On Windows:

    $ cd /path/to/this/directory
    $ ./run-windows.bat

Run game in web browser (see cgi-bin/README.md):

    $ cd /path/to/this/directory
    $ python3 cgi-bin/web_http.py
